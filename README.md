# RoadLens
RoadLens is a real-time traffic sign recognition system that uses a webcam to detect and interpret traffic signs.<br><br>
The system captures live video input through a webcam and processes each frame using computer vision and machine learning techniques. A trained model is used to classify the detected traffic sign into predefined categories such as speed limits, warnings, and mandatory signs. Once a sign is recognized, the system displays its meaning on the screen, allowing users to understand road instructions quickly and accurately.<br>
This project uses and combines technologies such as Python, OpenCV for image processing, and a deep learning model for classification.
